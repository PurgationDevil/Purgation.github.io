##  Jekyll 博客文件结构速查表

| 文件/文件夹                    | 在哪里找   | 改什么                                             |
| :----------------------------- | :--------- | :------------------------------------------------- |
| **`_config.yml`**              | 仓库根目录 | 网站标题、描述、作者、社交链接、URL                |
| **`index.html`** 或 `index.md` | 仓库根目录 | 首页布局、欢迎语、主页内容                         |
| **`_posts/`**                  | 仓库根目录 | **存放所有博客文章的地方**，每篇文章一个文件       |
| **`_layouts/`**                | 仓库根目录 | 页面布局模板（default.html, post.html, page.html） |
| **`_includes/`**               | 仓库根目录 | 可复用的组件（header, footer, 侧边栏）             |
| **`about.md`** 或 `about.html` | 仓库根目录 | "关于我"页面的内容                                 |
| **`css/`** 或 `assets/css/`    | 仓库根目录 | 网站样式、颜色、字体、背景                         |
| **`img/`** 或 `images/`        | 仓库根目录 | 头像、背景图、文章配图                             |
| **`CNAME`**                    | 仓库根目录 | 自定义域名（如果你有自己的域名）                   |
| **`404.html`**                 | 仓库根目录 | 页面不存在时显示的内容                             |

## `_posts/` —— 写文章的地方

**这是以后最常用的文件夹！**

**规则：**

- 文件名必须是 `YYYY-MM-DD-标题.md` 格式
- 例如：`2026-04-03-我的第一篇博客.md`

**每篇文章开头必须有这段"头信息"：**

```yaml
---
layout: post
title: "文章标题"
date: 2026-04-03
categories: 分类名称
tags: [标签1, 标签2]
---
```

头信息下面写正文，用 Markdown 语法。

## `_layouts/` —— 页面骨架

这里面放的是 HTML 布局模板：

| 文件           | 用途                                             |
| :------------- | :----------------------------------------------- |
| `default.html` | 所有页面的**基础框架**（header + footer + 主体） |
| `post.html`    | 单篇文章的布局                                   |
| `page.html`    | 普通页面（如"关于"页）的布局                     |

## `about.md` —— 自我介绍

这是你网站的"关于"页面。直接编辑里面的 Markdown 文字就行，不需要懂 HTML。

------

## `img/` —— 图片仓库

把头像、背景图、文章配图都放这里。

**改头像：** 找到 `img/` 里的头像文件（可能是 `avatar.jpg` 或 `logo.png`），用你的图片替换它（**文件名必须一样**）。

**改背景图：** 找到 `img/home-bg.jpg` 或类似的文件，替换掉。
